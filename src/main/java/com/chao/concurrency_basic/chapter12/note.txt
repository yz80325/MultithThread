1.main线程属于线程组main

2.一个线程创建的时候，如果没有指定ThreadGroup，则将该线程加入创建该线程的线程（父线程）所属的线程组

3.一个ThreadGroup可以拥有多个线程，也可以拥有多个线程组

4.在其他的线程组中（两个线程组不相关），其中一个线程组可以访问另一个线程组的相关信息

5.activeGroupCount()可以检测当前线程组下有多少个活动的线程组
	checkAccess()可以评估当前线程是否可以修改某个线程组	destroy()可以销毁某个线程组及其子线程和子线程组，但要保证被销毁的线程组内没有活动线程或线程组，否则抛出异常
	enumerate(Thread[] list, boolean recurse)可以复制线程组中的线程到list中，当recurse为true时，将线程组中的子线程组中的线程也一并复制
	
6.interrupt()会中断线程组内所有线程，包括子线程组中的所有线程

7.setDaemon()将线程组设置为守护线程组，当父线程组消亡时，它也会消亡，而守护线程组要等待线程组中最后一个线程死亡才会死亡，这时若要强制守护线程组死亡，需要显示调用destroy()
